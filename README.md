## Prerequisites

- [Node.js](https://nodejs.org/en/download/) (v12 or later)
- [npm](https://www.npmjs.com/get-npm) (v6 or later)
- [Nest CLI](https://docs.nestjs.com/cli/overview)

## Installation

```bash
git clone https://github.com/RiestelX/nest-rsa-api.git
```

```bash
cd nest-rsa-api
```

```bash
npm install
```

## Run

```bash
npm run start
```
